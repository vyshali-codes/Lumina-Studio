# Lumina Task Studio

## Overview
A modern, responsive To-Do List web application built using React, TypeScript, and Tailwind CSS. This project demonstrates core frontend development concepts including state management, DOM manipulation, and local storage persistence with a refined SaaS-like aesthetic.

## Features
- **Task Management**: Add, edit, delete, and complete tasks with a smooth UI.
- **Deadlines**: Optional deadlines with visual highlighting for overdue items.
- **Priority Levels**: High (red), Medium (yellow), and Low (green) indicators.
- **Categories & Tags**: Organize tasks by Work, Personal, Study, Health, or Other.
- **Recurring Tasks**: Support for daily and weekly recurrence with automatic regeneration.
- **Progress Tracking**: Real-time visual progress bar and completion percentage.
- **Analytics Dashboard**: Track your daily streak, completions today, and identify your most productive day.
- **Persistence**: All data is saved to `localStorage` and restored on reload.
- **Modern UI**: Handcrafted dark theme with lavender accents, glassmorphism effects, and fluid animations.

## Tech Stack
- **React 19**: Component-based architecture.
- **TypeScript**: Type-safe development.
- **Tailwind CSS 4**: Utility-first styling with custom theme variables.
- **Motion**: High-performance animations and transitions.
- **Lucide React**: Crisp, consistent iconography.

## Project Structure
- `src/App.tsx` → Main application logic and state management.
- `src/index.css` → Global styles and Tailwind theme configuration.
- `src/main.tsx` → Entry point.
- `metadata.json` → App metadata and permissions.

## How to Run
1. The application is automatically served via Vite.
2. Open the provided App URL in your browser.
